## ğŸ”§ Troubleshooting

### Camera not working?
1. Check browser permissions (click lock icon in address bar)
2. Reload page
3. Try different browser (Chrome works best)

### Voice detection not working?
1. Check microphone permission
2. Speak louder and clearer
3. Reduce background noise
4. Try Chrome or Edge (Safari has limited support)

### OCR not accurate?
1. Retake photo with better lighting
2. Hold phone steady
3. Make sure label is flat
4. Try uploading instead of camera

### Notifications not appearing?
1. Check browser notification settings
2. Make sure notifications are enabled in OS
3. Try different browser
4. Don't use incognito mode
